        Configuratie certificaat in CA

        In deze deelopdracht maken we een certificaat aan via de certificaataanvraag uit de vorige opdracht.
        We exporteren ook het certificaat uit de CA.


        • Log in op de server SRV-WEB-01 met de domain administrator van het netwerk AD.Training.com
        • Open de console 'Certification Authority'
        • Rechtsklik op 'SRV-WEB-01' en kies onder 'All Tasks' voor "Submit new request..."
          • Browse naar de net aangemaakte request file "certreq.txt" op de C-schijf
        • Klik nu op de map 'Pending Requests'
        • Rechtsklik op de request en kies onder 'All Tasks' voor "Issue"
        De request wordt nu omgezet naar een certificaat.
        • Klik nu op de map 'Issued Certificates'
        • Rechtsklik op het certificaat en kies voor "Open"
          • Ga naar het tabblad 'Details' en klik op de knop "Copy to File..."
            • Klik op Next
            • Bij 'Export File Format' kies je voor de standaardinstelling en klik je op Next
            • Bij 'File to Export' geef je als file name "C:\secure.cer" mee en klik je op Next
            • Klik op Finish om de wizard af te sluiten
        Er verschijnt een venster met de boodschap "Export was successful"
